== Description ==

Make your feature box useful. This plugin adds a box inside all Post & Page edit pages that allows you to put content inside your feature box.

== Installation ==

* Install plugin
* Go to Thesis Options > Design Options and set the Feature Box to display site wide.
* Go to a page or a post and add your custom content/code into the Feature Box field.
